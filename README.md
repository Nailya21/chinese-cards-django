# Chinese Cards Django

Учебный веб-проект на Django для изучения китайских слов с помощью карточек.

## Автор
Сафина Наиля Айратовна, М02-503пг

## Описание проекта

Приложение позволяет:
- просматривать список карточек с китайскими словами;
- открывать отдельную карточку;
- добавлять новые карточки;
- редактировать существующие карточки;
- фильтровать карточки по теме и уровню HSK;
- использовать страницу тренировки для повторения слов.

Проект выполнен в рамках учебного задания по разработке web-приложения на Django.

## Функциональность

- Главная страница
- Список карточек
- Страница одной карточки
- Добавление карточки
- Редактирование карточки
- Валидация формы
- Фильтрация по теме и HSK
- Страница тренировки

## Технологии

- Python
- Django
- SQLite
- HTML
- CSS

## Как запустить проект локально

### 1. Предварительные требования

На компьютере должны быть установлены:
- Python 3.12 или новее
- Git
- Visual Studio Code или другой редактор кода

### 2. Клонирование репозитория

Открыть командную строку и выполнить:

```bash
git clone https://github.com/Nailya21/chinese-cards-django.git

### 3. Переход в директорию проекта

Перейдите в папку с проектом командой:

cd chinese-cards-django
### 4. Создание виртуального окружения

Для изоляции зависимостей проекта необходимо создать виртуальное окружение:

py -m venv .venv

После выполнения команды в папке проекта будет создана директория .venv.

### 5. Активация виртуального окружения

Для Windows виртуальное окружение активируется следующей командой:

.venv\Scripts\activate

После активации в начале строки командной строки должно появиться обозначение виртуального окружения, например:

(.venv)
### 6. Установка зависимостей

После активации виртуального окружения необходимо установить зависимости проекта:

py -m pip install --upgrade pip
py -m pip install Django pylint
### 7. Применение миграций базы данных

Для создания необходимых таблиц в базе данных выполните команду:

py manage.py migrate
### 8. Создание учетной записи администратора

Для доступа к административной панели Django создайте учетную запись администратора:

py manage.py createsuperuser

После запуска команды система последовательно запросит:

имя пользователя;
адрес электронной почты;
пароль.

Введенные данные необходимо запомнить, так как они потребуются для входа в административную панель.

### 9. Запуск локального сервера

Для запуска проекта выполните команду:

py manage.py runserver

После успешного запуска сервер будет доступен по адресу:

http://127.0.0.1:8000/
### 10. Открытие проекта в браузере

После запуска сервера можно открыть в браузере следующие страницы:

главная страница проекта: http://127.0.0.1:8000/
список карточек: http://127.0.0.1:8000/cards/
форма добавления карточки: http://127.0.0.1:8000/cards/add/
страница массового добавления карточек: http://127.0.0.1:8000/cards/bulk-add/
режим тренировки: http://127.0.0.1:8000/study/
административная панель Django: http://127.0.0.1:8000/admin/

### 11. Первичное заполнение проекта данными

После первого запуска проект может не содержать карточек. Для начала работы можно:

войти в административную панель Django и вручную добавить несколько карточек;
воспользоваться формой добавления карточек на сайте;
использовать страницу массового добавления карточек через таблицу.

После добавления данных становятся доступны:

просмотр карточек;
фильтрация по теме и уровню HSK;
режим тренировки;
подсчёт статистики прохождения.

### 12. Повторный запуск проекта

При повторной работе с проектом необходимо:

открыть командную строку;
перейти в папку проекта;
активировать виртуальное окружение;
запустить сервер.

Команды для повторного запуска:

cd chinese-cards-django
.venv\Scripts\activate
py manage.py runserver

### 13. Остановка сервера

Для остановки локального сервера в командной строке необходимо нажать сочетание клавиш:

Ctrl + C

### 14. Проверка качества кода

Для запуска проверки качества кода с помощью pylint необходимо выполнить:

pylint cards config manage.py
